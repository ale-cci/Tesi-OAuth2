# Tesi Oauth
> Working Progress

##### Dependencies
* docker-compose: start jupyter notebook presentation
* xelatex: compile pdf

##### Start presentation
```sh
$ docker-compose up --build -d      # Start notebook
$ docker-compose logs notebook      # Copy access token login in via web
```
